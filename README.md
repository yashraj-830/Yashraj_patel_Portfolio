# Yashraj Singh Patel Portfolio

A fully static, responsive portfolio website for Yashraj Singh, a Python Django Developer with 1+ year of experience.

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React Icons

## Features

- Dark futuristic developer-focused design
- Animated network and particle background
- Responsive navbar with mobile menu
- Hero, About, Skills, Projects, Learning Journey, Services, GitHub Activity, and Contact sections
- Static placeholder links for resume, GitHub, LinkedIn, email, project demos, and screenshots
- Visual-only contact form with no backend, API, or submission handling

## Commands

```bash
npm install
npm run dev
npm run build
npm run lint
```

## Notes

This is a static frontend-only portfolio. Replace placeholder values in `src/data/portfolio.ts` with real profile links, contact details, resume path, and project links when available.
